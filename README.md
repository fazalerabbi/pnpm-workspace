## Installation

Instructions on how to set up and run the project locally.

```bash
# Clone the repository
git clone git@github.com:fazalerabbi/pnpm-workspace.git

# Navigate to the project directory
cd pnpm-workspace

# Install dependencies
pnpm install

# Start the project
pnpm start
